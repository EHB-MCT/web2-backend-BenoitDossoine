Welcome to the gamenight API!

Routes:

GET /users
Returns a list of all users
GET /user/:userId/boardgames
Returns a list of all boardgames of user with id 'userId'
POST /user/:userId/boardgames
Add a boardgame to the collection of user with id 'userId'
DELETE /user/:userId/boardgames/:boardgameId
Delete boardgame with id 'boardgameId' from the collection of user with id 'userId'
GET /boardgames
Returns a list of all boardgames that have ever been added to the database
POST /boardgames
Add a boardgame to the general library
GET /boardgames/:boardgameId
Return the boardgame with id 'boardgameId'
GET /categories
Return all existing categories
GET /categories
Return all existing categories
POST /categories
Update all existing categories
GET /gamenights
Return all existing gamenights
POST /gamenights
Create a gamenight with given parameters
DELETE /gamenights/:gamenightId
Delete gamenight with id 'gamenightId'
GET /user/:userId/gamenights
Return all existing gamenights from user with id 'userId'
GET /user/:userId/gamenights/:gamenightId
Return gamenight with id 'gamenightId' from user with id 'userId'
